Parameters
pvParametersPointer to task parameters.
Returns
n/a

This task sends data to a queue for TaskRT4() and waits for the FFT task to signal completion. It calculates the total wall clock time for 5 FFTs and prints the result to the serial monitor. After that, it suspends itself.
